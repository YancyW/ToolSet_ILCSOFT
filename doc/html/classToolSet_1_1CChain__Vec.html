void ToolSet::CChain_Vec::Clear (  ) 

Clear Notice: Remember to Clear the chain after use it.

If you use combined_RC command, it will build a "new" Particle. So you need to delete this point with Clear() function. I didn't write Clear() into deconstructor because it may cause some trouble when transfer the chain to other functions.
